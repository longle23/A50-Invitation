🎉 Demo Hệ thống QR Check-in

📋 Hướng dẫn sử dụng:

  1. Khởi động server: npm start
  2. Tạo QR codes: npm run generate-qr
  3. Truy cập trang quản lý: http://localhost:3000
  4. Test check-in: Click vào các link demo bên dưới

🛠️ API Endpoints:

GET / - Trang quản lý chính
GET /checkin/:guestId - Trang check-in cá nhân hóa
POST /api/checkin/:guestId - API check-in
GET /api/checkins - Lấy danh sách check-in
GET /api/export - Xuất báo cáo

🚀 Cách triển khai:

  1. In QR codes từ thư mục output/qrs/
  2. Phân phối cho khách mời
  3. Khách quét QR → mở trang check-in
  4. Nhấn "Check In" → hệ thống ghi nhận
  5. Theo dõi tại http://localhost:3000
